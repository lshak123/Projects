# 백화점 고객분석을 통한 마케팅 전략 제시

<br/>

## 프로젝트 기간
- 2022.06(1개월)

<br/>

## 프로젝트 소개
PYGAME 모듈을 활용해 총알 피하기 게임을 만들어보자.
 
<br/>

## 프로젝트 과정

<br/>
#### 총 11개의 미션을 한단계씩 추가해나가는 식으로 구현
1. 비행기 효과음 발생
2. 총알에 맞았을 때 터지는 그림 발생
3. 배경그림이 같이 움직일 수 있도록 바꾸기
4. 비행기에 목숨개념을 도입
5. 무적시간 추가 및 무적시간동안 반짝거리게 변경
6. 남은 생명력을 막대기와 숫자로 표현
7. 총알을 여러개 생성
8. 총알마다 다른 데미지
9. 최종 결과에 대한 순위 기록
10. 게임오버시, 생존시간 출력
11. 게임오버시, 신기록에 대해 강조해서 출력
